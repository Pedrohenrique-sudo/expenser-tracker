📖 Sobre o Projeto
O ExpenseTracker foi desenvolvido para quem busca controle financeiro sem complicações. Diferente de outros apps, ele roda inteiramente no cliente, garantindo que seus dados nunca saiam do seu navegador.

## ✨ Funcionalidades Principais
* **📊 Dashboards:** Gráficos de linha e pizza interativos.
* **💼 Orçamentos:** Defina limites mensais por categoria.
* **🎯 Metas:** Planeje suas economias com barras de progresso.
* **🌗 Dark Mode:** Suporte completo para modo escuro e claro.
* **💾 Local Storage:** Seus dados ficam salvos apenas no seu navegador.

## 🚀 Como começar
### Instalação
```bash
# Clone o repositório
git clone [https://github.com/Pedrohenrique-sudo/Tracker-.git](https://github.com/Pedrohenrique-sudo/Tracker-.git)
# Instale as dependências
pnpm install
# Inicie o projeto
pnpm dev

# caso de erro na instalação, esses comandos abaixo devem resolver
#coloque um de cada vez 

pnpm approve-builds @tailwindcss/oxide esbuild

pnpm add -D lightningcss-android-arm64

pnpm dev
